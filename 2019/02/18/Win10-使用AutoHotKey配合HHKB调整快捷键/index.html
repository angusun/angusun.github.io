Win10 - 使用AutoHotKey配合HHKB调整快捷键
写于2019年02月18日

按照之前的使用习惯捣鼓了一份配置。大致如下：

主要实现LCtrl + wasd 控制方向键，LCtrl + qe 实现行首行尾跳转，LCtrl + fj 实现词前词后跳转，LCtrl + gh 上下移动当前行，LCtrl + ru/ty/vn 往前往后/首尾/词前词后的选中。 还有就是Alt 代替 Ctrl 进行了保存，复制，粘贴，撤销，剪切等操作。以及右ALT + 数字 实现 FN键。
